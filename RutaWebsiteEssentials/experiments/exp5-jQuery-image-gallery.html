jQuery Image Gallery SlideShow

Paintings by Milind Mullick


About the experiment

In this experiment I have implemented an image-gallery using jQuery. It also uses z-index of CSS which specifies the stack order of an element. It works on positioned elements and hence, the image having highest index is displayed first.

Code Snippet

#slideshow img.active{
z-index:10;
}
#slideshow img.last-active{
z-index:9;
}

function slideshow() {
var $active = $('#slideshow img.active');
var $next = $active.next();

$next.addClass('active');
$active.removeClass('active');
}
$(function () {
setInterval("slideshow()", 3000);
});
