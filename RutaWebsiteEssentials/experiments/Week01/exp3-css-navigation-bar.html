Experiment Description

This is a navigation bar having submenus. Hovering on the menu will display submenus of the selected menu. The experiment is implemented using CSS.


Code Snippet

#horizontalmenu ul {
list-style-type: none;
margin: 0;
padding: 0;
overflow: hidden;
}

#horizontalmenu li {
float: left;
}

#horizontalmenu a:hover {
color:#b21056;
background-color:bisque;
}

#horizontalmenu li ul{
background-color:rgba(99, 52, 159, 0.75);
color:bisque;
display:none;
}

#horizontalmenu li:hover ul{
display:block;
}

#horizontalmenu li ul li{
clear:both;
display:block;
}

a:link, a:visited {
display: block;
width: 150px;
height: 30px;
background-color: rgba(99, 52, 159, 0.75);
color:bisque;
text-decoration:none;
text-align:center;
}
