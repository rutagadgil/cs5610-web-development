CSS : Animations




Experiment description

In this experiment I have implemented the animation property using CSS. An animation allows you to gradually change the style of an element.

Code Snippet

-webkit-animation: myanimation 10s infinite;

@-webkit-keyframes myanimation {
from {
left: 100px;
background:red;
-webkit-transform: rotate(0deg);
}
to {
left: 600px;
background:blue;
-webkit-transform: rotate(360deg);
}
