Experiment Description

In this experiment I have implemented jQuery AJAX. AJAX is used to exchange data with a server, and update parts of a web page - without reloading the whole page. Query provides several methods for AJAX functionality. With the jQuery AJAX methods, you can request text, HTML, XML, or JSON from a remote server using both HTTP Get and HTTP Post - And you can load the external data directly into the selected HTML elements of your web page. Here, I have used the Get method to get data from a couple of my previous experiments.
