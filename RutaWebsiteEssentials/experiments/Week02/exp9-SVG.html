A Ball
A Ball with its shadow

Experiment description

In this experiment I have implemented HTML5 SVG elements. Here, RadialGradient is used to create the effect for the Ball and LinearGradient is used to create effect for the shadow. SVG is an acronym for Scalable Vector Graphics and is used to define vector based graphics for the Web. SVG graphics do not lose any quality if they are zoomed or resized and every element and attribute in SVG files can be animated.

Code Snippet


<div id="ball-with-shadow">
<svg width="500px" height="200px">
<!--creating a gradient for ball's shadow-->
<defs>
<linearGradient id="grad1" x1="0%" y1="0%" x2="100%" y2="0%">
<stop offset="10%" style="stop-color:black;stop-opacity:1" />
<stop offset="100%" style="stop-color:#808080;stop-opacity:1" />
</linearGradient>
</defs>
<!--actual shadow-->
<ellipse cx="300" cy="172" rx="75" ry="15" fill="url(#grad1)" />
<!--ball-->
<defs>
<radialGradient id="grad2" cx="50%" cy="50%" r="50%" fx="25%" fy="25%">
<stop offset="0%" style="stop-color:rgb(255,255,255);stop-opacity:0.95" />
<stop offset="100%" style="stop-color:rgb(0,0,255);stop-opacity:1" />
</radialGradient>
</defs>
<circle cx="245" cy="140" r="40" stroke="blue" fill="url(#grad2)"></circle>
</svg>
</div>
