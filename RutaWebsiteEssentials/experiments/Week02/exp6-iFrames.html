Experiment Description

This is a continuation of the navigation bar experiment. Hovering on the menu will display submenus of the selected menu. Selecting a menu will display content related to the submenu in an iFrame. The experiment is implemented using CSS.


Code Snippet


<script type="text/javascript">
function changeIframe(url) {
document.getElementById('iFrame').src = url;
}
</script>


<li><a href="#home" onclick="changeIframe('iFramePages/home.html');">Home</a></li>
<li>
<a href="#indianPaintings">Indian Paintings</a>
<ul>
<li><a href="#Madhubani" onclick="changeIframe('iFramePages/Madhubani.html');">Madhubani</a></li>
<li><a href="#Miniature" onclick="changeIframe('iFramePages/Miniature.html');">Miniature</a></li>
