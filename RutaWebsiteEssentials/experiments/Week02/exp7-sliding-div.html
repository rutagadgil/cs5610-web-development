Experiment Description

This is an experiment about sliding divs. On hovering over "About Watercolor painting", a sliding div appears which contains information about Watercolor painting. The effect is acheived using CSS transitions and slideToggle() function in jQuery.


Code Snippet

$(document).ready(function () {
$(".trigger").hover(function () {
$("#panel").slideToggle("slow");
$(this).toggleClass("active");
return false;
});
});
