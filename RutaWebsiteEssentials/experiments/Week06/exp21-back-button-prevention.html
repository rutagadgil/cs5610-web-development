Back button prevention

Experiment Description

In this experiment, I have implemented previous page navigation prevention. This is achieved using window.history.forward(). We cannot literally disable the back button. However, this is a good method to prevent such navigations. This technique can be used in a variety of applications especially after form submissions or logouts and is a very important issue that needs to be addressed for security purposes. It could also be very useful for my project implementation. Hence, I have included it as an experiment.

Code Snippet

$(document).ready(function () {
$("#nextPage").click(function () {
window.location.href = "newPage.html";
});
});
window.history.forward();
