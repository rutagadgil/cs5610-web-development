jQuery - Binding & Unbinding

Click here



Experiment Description

This experiment is based on the bind(), unbind() properties of jQuery. Binding is the coupling of an handler to an element, that will run when an event occurs on said element. Unbinding is the de-coupling of an handler from an element, so that when an event occurs the handler function will no longer run.


Code Snippet

$("#unbindBtn").bind("click",(function () {
$('#bindingDiv').unbind("click");
$("#contents").text("");
}));

$("#bindBtn").bind("click", (function () {
$("#bindingDiv").bind({
click : function(){
$("#contents").append("This div is now binded! ");
}
});
}));
